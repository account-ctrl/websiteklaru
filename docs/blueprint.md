# **App Name**: KlaroGov Landing Page

## Core Features:

- Hero Section with Floating UI Stack: Visually engaging hero section featuring a blurred aurora background and a floating UI stack of cards with subtle animations.
- Capabilities Grid (Bento Style): Display key capabilities in a bento-style grid layout with interactive animations on hover.
- Dual Experience Toggle: Showcase the dual admin/resident experiences with a toggle switch that transitions between abstract wireframe visualizations.
- Tailwind CSS Styling: Use Tailwind CSS to style all the components and visuals.
- Framer Motion Animations: Use Framer Motion for creating smooth animations, layout transitions, and floating effects.
- SVG Icons: Implement SVG icons from Lucide-React for sharp and scalable visuals.

## Style Guidelines:

- Primary Background (Canvas): #F8FAFC (Light Blue-Grey) - Matches the dashboard background.
- Header/Hero Background: #1E293B (Dark Slate) - Matches the Admin Header.
- Headings Text: #1E293B (Dark Slate).
- Body Text: #64748B (Medium Grey).
- Functional Accent - Cyan (Documents/Requests): #06B6D4.
- Functional Accent - Rose Red (Emergency/Alerts): #F43F5E.
- Functional Accent - Amber (News/Community): #F59E0B.
- Functional Accent - Slate (ID/Official): #334155.
- Body and headline font: 'Inter' sans-serif for a modern, machined look.
- Use Lucide-React SVG icons.
- CSS Grid layout for the Capabilities section (bento style).
- Utilize CSS shapes and gradients.
- Implement Framer Motion animations for transitions and effects.
